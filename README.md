# simple_banking_program
Description

This project is a simple command-line banking application developed in Python. It allows users to perform basic banking operations such as checking their balance, depositing money, and withdrawing funds through a menu-driven interface.

Features

* View current account balance
* Deposit funds into the account
* Withdraw funds with balance validation
* Input validation for incorrect amounts
* Simple and easy-to-use command-line menu

Technology Used

* Python 3

How to Run

1. Clone the repository:

```
git clone https://github.com/yourusername/simple-banking-program.git
```

2. Navigate to the project directory:

```
cd simple-banking-program
```

3. Run the program:

```
python main.py
```

Example Menu

```
Banking Program
1. Balance
2. Deposit
3. Withdraw
4. Exit
```

Purpose

The purpose of this project is to practice fundamental programming concepts in Python, including functions, loops, conditional statements, and user input handling.
